# Minecraft Items

このリポジトリには、Minecraftのアイテムテクスチャ画像が含まれています。

## 内容

アイテムはカテゴリー別に分類されています:

- `blocks/` - ブロック類（建築ブロック、鉱石、装飾ブロックなど）- 812ファイル
- `foods/` - 食料品（果物、肉、魚、料理など）- 42ファイル
- `equipment/` - 装備品（防具、武器、道具、馬具など）- 103ファイル
- `other_items/` - その他のアイテム（素材、ポーション、スポーンエッグ、レコードなど）- 532ファイル

## ファイル数

合計: 1,489個のアイテムテクスチャ

## 使用方法

各アイテムの画像は対応するカテゴリーフォルダ内にあります。
ファイル名はMinecraftのアイテムIDに対応しています。

例:
- `equipment/diamond_sword.png`
- `equipment/iron_pickaxe.png`
- `foods/golden_apple.png`
- `blocks/stone.png`
